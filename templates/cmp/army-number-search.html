This search tool can match army numbers from the 1920's to the end of World War 2 which was based on the regiment that a soldier joined initially, and the post war numbering system which is army wide. Pre 1920's, soldiers had regimental numbers so this site cannot be used for matching those. More about army number can be found in this article. A search for the number 7875698 should return Royal Tank Regiment. It is the number assigned to T.E. Lawrence who joined the Tank Corps in March 1923.

Enter an army number to find out which regiment block the number was assigned from (upto WW2) or approximately when the soldier joined the army (post WW2).
